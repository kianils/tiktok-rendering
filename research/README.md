# Research / ML (optional)

The main Next.js app is intentionally **local, rule-based, and export-grounded** so the Zuboff framing stays honest.

If you train models on **public datasets**, keep that work here (notebooks, Python env, batch jobs) and **do not** treat learned weights as TikTok’s internal recommender. Ship separate copy and filenames so users never confuse:

- **Generic / public-data models** (research artifacts), versus  
- **Personal interpretation** from their official **Download your data** ZIP (the default product path).

Review dataset **license, provenance, and ethics** before publishing anything derived from third-party corpora.
